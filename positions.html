Values for the Position Property
Position Description
static Positions according to flow of the page without special properties (It is the default position of all HTML elements and is not affected by top, bottom, right, left properties.)
relative Positions elements relative to where they would appear in the page rendering within the browser window, regardless of screen size
fixed Keeps elements in place in the viewport, or page rendering within the browser window, regardless of page scrolling
sticky Switches an element's position to either fixed or relative depending on how far the user has scrolled on the page
absolute Positions element relative to its parent (The parent must have a position property set that is not static or the element will find the next parent element.)
inherit Assigns the element the value of its parent
initial Assigns the element what its default value should be
